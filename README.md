# GOMANGA #


Manga Crawler app written in golang


## Purpose ##

Learning Golang by doing familiar things 



## Usage ##

So far, you can build with 

``` go build main.go -o . ```

And then you can run 

``` gomanga -s "mangareader" -m "bleach" -c 100"```
